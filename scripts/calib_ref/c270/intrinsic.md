## 물리 검증 체크리스트

> 핸드-아이 수집 전에 아래 항목으로 인트린직 결과가 실제와 맞는지 확인한다.

### 방법: ArUco 마커 거리 비교

1. **ID 0 마커(5cm)를 카메라 정면 약 30cm 위치에 수직으로 세운다**
2. `python3 scripts/c270_handeye_collect.py` 실행 (캡처 모드 진입)
3. 화면에 표시되는 `dist_m` 값이 **실측 거리와 ±5mm 이내**인지 확인
4. 마커를 50cm 위치로 옮기고 동일하게 확인

### 체크 항목

- [ ] 30cm 위치: 화면 거리 = `0.30 ± 0.005 m`
- [ ] 50cm 위치: 화면 거리 = `0.50 ± 0.005 m`
- [ ] 마커 윤곽선(초록 사각형)이 실제 마커 테두리와 잘 일치
- [ ] 마커를 기울였을 때 축(axis) 방향이 실제와 일치

### 판정 기준

| 결과 | 조치 |
|------|------|
| 거리 오차 < 5mm | ✅ 통과 → 핸드-아이 수집 진행 |
| 거리 오차 5~10mm | ⚠️ 마커 실측 크기 재확인 후 재판정 |
| 거리 오차 > 10mm | ❌ `c270_intrinsic_calib.py` 재실행 (샘플 30장 이상 권장) |
