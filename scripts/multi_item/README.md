# multi_item — 캘리브레이션·인식 소모/활용 아이템 관리

> 인쇄·제작해서 실제로 사용하는 물리 아이템의 규격·이미지·용도를 관리한다.
> 코드에서 아이템 관련 경로·규격을 참조할 때 이 디렉토리를 기준으로 한다.

## 구조

```
multi_item/
├── markers/        ← ArUco 마커 (인쇄 PNG/PDF + 규격)
└── checkerboard/   ← 체커보드 패턴 (인트린직 캘리브레이션용)
```

## 아이템 목록

| 아이템 | 디렉토리 | 현재 상태 | 용도 |
|--------|----------|----------|------|
| ArUco 마커 ID 0·1·2 | `markers/` | ✅ 인쇄본 있음 | 탑뷰 캘리브, C270 eye-in-hand 캘리브, 공구함 위치 인식 |
| 체커보드 9×6 / 25mm | `checkerboard/` | 📋 규격만 (인쇄 필요 시 생성) | C270 인트린직 캘리브레이션 |
